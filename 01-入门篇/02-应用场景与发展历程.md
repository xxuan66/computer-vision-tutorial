# 第二章：计算机视觉应用场景与发展历程

> 从典型场景出发，理解 CV 为什么重要，以及它是怎样一步步发展到今天的

## 2.1 医疗影像分析

### 2.1.1 应用场景

| 应用 | 说明 | 技术 |
|------|------|------|
| **X 光片分析** | 肺炎、骨折检测 | 图像分类 |
| **CT 影像** | 肿瘤识别、器官分割 | 目标检测、分割 |
| **MRI 分析** | 脑部疾病诊断 | 3D 分割 |
| **病理切片** | 癌症细胞检测 | 高分辨率分类 |
| **眼底图像** | 糖尿病视网膜病变 | 分类 + 分割 |

### 2.1.2 典型案例

**Google Health - 乳腺癌筛查**

```
技术：深度学习 CNN
数据集：28,000+ 乳腺 X 光片
准确率：超越人类放射科医生
- 假阳性率降低：5.7%
- 假阴性率降低：9.4%
```

**一个典型工作流程：**
```
乳腺 X 光片 → 预处理 → CNN 分类 → 结果分析
                ↓           ↓
          去噪增强    良性/恶性
```

### 2.1.3 挑战与机遇

**挑战：**
- 数据隐私保护
- 标注成本高（需要医生）
- 模型可解释性要求高
- 法规审批严格

**机遇：**
- 医疗资源不均衡
- 医生工作量大
- 早期诊断需求
- AI 辅助诊断趋势

## 2.2 自动驾驶

### 2.2.1 核心技术栈

```
传感器 → 感知 → 决策 → 控制
  ↓       ↓      ↓      ↓
摄像头  目标检测  路径规划  转向/加速
激光雷达 语义分割  行为预测  刹车
毫米波  深度估计  场景理解
```

### 2.2.2 CV 在自动驾驶中的应用

| 任务 | 技术 | 代表模型 |
|------|------|---------|
| **车道线检测** | 语义分割 | LaneNet |
| **交通标志识别** | 图像分类 | ResNet |
| **行人检测** | 目标检测 | YOLO, Faster R-CNN |
| **障碍物识别** | 3D 检测 | PointRCNN |
| **场景分割** | 全景分割 | Mask2Former |
| **深度估计** | 单目深度 | MiDaS |

### 2.2.3 典型架构

**Tesla Autopilot：**
```
8 个摄像头 → Bird's Eye View → Occupancy Network → 规划控制
              ↓
        BEVFormer
```

**Waymo：**
```
激光雷达 + 摄像头 → 多模态融合 → 预测 → 规划
```

### 2.2.4 技术挑战

- 极端天气（雨雪雾）
- 长尾场景（罕见情况）
- 实时性要求（<100ms）
- 安全性（99.9999% 可靠）

## 2.3 安防监控

### 2.3.1 应用场景

| 场景 | 功能 | 技术 |
|------|------|------|
| **人脸识别** | 门禁、考勤 | FaceNet, ArcFace |
| **行为分析** | 异常检测 | 动作识别 |
| **人流统计** | 密度估计 | 计数网络 |
| **车辆识别** | 车牌、车型 | OCR+ 分类 |
| **轨迹跟踪** | 多目标跟踪 | DeepSORT |

### 2.3.2 智慧小区案例

**系统架构：**
```
摄像头 → 人脸检测 → 特征提取 → 比对 → 门禁控制
           ↓           ↓          ↓
        MTCNN    FaceNet    特征库
```

**性能指标：**
- 识别速度：< 0.3 秒
- 准确率：> 99%
- 支持：活体检测（防照片攻击）
- 容量：10 万 + 人脸库

### 2.3.3 隐私保护

**技术方案：**
- 人脸脱敏（模糊处理）
- 数据加密存储
- 访问权限控制
- 数据定期删除

## 2.6 计算机视觉发展历程

### 2.6.1 传统方法阶段（1960s-2012）

**特点：** 手工设计特征 + 浅层模型

| 年代 | 代表方法 | 贡献 |
|------|---------|------|
| **1960s** | Roberts 边缘检测 | 边缘提取开端 |
| **1970s** | SIFT 特征 | 尺度不变特征 |
| **1980s** | 形态学操作 | 图像后处理 |
| **1990s** | Viola-Jones | 实时人脸检测 |
| **2000s** | HOG+SVM | 行人检测 SOTA |

**典型流程：**
```
图像 → 预处理 → 特征提取 → 分类器 → 结果
          ↓           ↓          ↓
       滤波去噪    HOG/SIFT   SVM/AdaBoost
```

### 2.6.2 深度学习阶段（2012-至今）

**里程碑事件：**

| 年份 | 事件 | 意义 |
|------|------|------|
| **2012** | AlexNet 赢得 ImageNet | 深度学习元年 |
| **2014** | VGG/GoogLeNet | 网络更深更宽 |
| **2015** | ResNet | 残差连接，千层网络 |
| **2016** | YOLO | 实时目标检测 |
| **2017** | Transformer | 注意力机制 |
| **2020** | ViT | Vision Transformer |
| **2022** | Stable Diffusion | 图像生成普及 |
| **2024** | Sora | 文生视频 |

**准确率提升（ImageNet）：**
```
2010: 72% (传统方法)
2012: 85% (AlexNet) ← 转折点
2015: 96% (ResNet)
2020: 98%+ (EfficientNet)
人类：约 95%
```

### 2.6.3 技术演进趋势

**网络架构：**
```
浅层网络 → 深层网络 → 残差网络 → Transformer
(LeNet)   (VGG-19)   (ResNet)    (ViT)
```

**学习方式：**
```
监督学习 → 弱监督 → 自监督 → 多模态
```

**应用场景：**
```
单一任务 → 多任务 → 通用视觉 → AGI
```
