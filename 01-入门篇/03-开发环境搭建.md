## 3.1 Python 环境配置

### 3.1.1 安装 Python

**推荐版本：Python 3.8+**

**macOS：**
```bash
# 使用 Homebrew 安装
brew install python@3.9

# 验证安装
python3 --version  # Python 3.9.x
pip3 --version     # pip 21.x
```

**Ubuntu/Debian：**
```bash
# 更新包列表
sudo apt update

# 安装 Python
sudo apt install python3.9 python3.9-venv python3.9-dev

# 验证安装
python3 --version
pip3 --version
```

**Windows：**
```bash
# 从官网下载安装
https://www.python.org/downloads/

# 安装时勾选 "Add Python to PATH"

# 验证安装
python --version
pip --version
```

### 3.1.2 创建虚拟环境

**为什么需要虚拟环境？**
- ✅ 隔离项目依赖
- ✅ 避免版本冲突
- ✅ 方便部署和分享

**创建步骤：**
```bash
# 创建项目目录
mkdir cv-tutorial
cd cv-tutorial

# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境

# macOS/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate

# 验证激活成功（命令行前面应该有 (venv)）
which python  # macOS/Linux
where python  # Windows
```

## 3.5 第一个 CV 程序

### 3.5.1 读取并显示图像

```python
import cv2
import matplotlib.pyplot as plt

# 读取图像
img = cv2.imread('image.jpg')

# 转换颜色空间（BGR → RGB）
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

# 显示图像
plt.figure(figsize=(10, 8))
plt.imshow(img_rgb)
plt.axis('off')
plt.title('My First CV Program')
plt.show()

print(f"图像形状：{img.shape}")
print(f"高度：{img.shape[0]}")
print(f"宽度：{img.shape[1]}")
print(f"通道数：{img.shape[2]}")
```

### 3.5.2 图像基本操作

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg')

# 1. 调整大小
resized = cv2.resize(img, (256, 256))

# 2. 转换为灰度图
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 3. 图像翻转
flipped = cv2.flip(img, 1)  # 1 表示水平翻转

# 4. 保存图像
cv2.imwrite('output.jpg', img)
cv2.imwrite('gray.jpg', gray)

print("✅ 图像操作完成！")
```

### 3.5.3 摄像头调用

```python
import cv2

# 打开摄像头（0 表示默认摄像头）
cap = cv2.VideoCapture(0)

while True:
    # 读取帧
    ret, frame = cap.read()
    
    if not ret:
        break
    
    # 显示帧
    cv2.imshow('Camera', frame)
    
    # 按 q 键退出
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# 释放资源
cap.release()
cv2.destroyAllWindows()
```

## 3.6 数据集下载

### 3.6.1 MNIST（手写数字）

```python
from torchvision import datasets, transforms

# 下载 MNIST 数据集
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.1307,), (0.3081,))
])

train_dataset = datasets.MNIST(
    root='./data',
    train=True,
    download=True,
    transform=transform
)

test_dataset = datasets.MNIST(
    root='./data',
    train=False,
    download=True,
    transform=transform
)

print(f"训练集：{len(train_dataset)} 张图像")
print(f"测试集：{len(test_dataset)} 张图像")
```
