## 1.1 什么是计算机视觉

### 1.1.1 定义

**计算机视觉（Computer Vision, CV）** 是一门研究如何让计算机“看懂”图像和视频的学科。

> 简单说，就是让计算机像人类一样，能够识别和理解图像中的内容。

### 1.1.2 核心任务

计算机视觉主要解决三大任务：

| 任务 | 说明 | 例子 |
|------|------|------|
| **图像分类** | 判断图像是什么 | "这是一只猫" |
| **目标检测** | 找出物体在哪里 | "猫在图像左上角" |
| **图像分割** | 像素级分类 | "这些像素属于猫" |

**图示：**
```
输入图像 → [分类：猫]
         → [检测：猫 (x1,y1,x2,y2)]
         → [分割：猫的像素掩码]
```

### 1.1.3 与人类视觉的对比

| 能力 | 人类视觉 | 计算机视觉 |
|------|---------|-----------|
| **识别速度** | 约 13 毫秒 | 毫秒到秒级 |
| **准确率** | 约 95% | 最高 99%+ |
| **适应性** | 极强 | 需要训练 |
| **疲劳度** | 会疲劳 | 不疲劳 |
| **一致性** | 有波动 | 完全一致 |

## 1.2 计算机视觉的应用

### 1.2.1 医疗影像

**应用场景：**
- X 光片分析（肺炎检测）
- CT 影像分割（肿瘤识别）
- 病理切片分析（癌症诊断）
- 眼底图像分析（糖尿病视网膜病变）

**典型案例：**
```
Google Health - 乳腺癌筛查
- 准确率：超越人类放射科医生
- 假阳性率：降低 5.7%
- 假阴性率：降低 9.4%
```

### 1.2.2 自动驾驶

**应用场景：**
- 车道线检测
- 交通标志识别
- 行人检测
- 障碍物识别
- 语义分割（道路、车辆、行人）

**技术栈：**
```
摄像头 → 图像预处理 → 目标检测 → 决策规划
            ↓
        深度学习模型（YOLO, Faster R-CNN）
```

### 1.2.3 安防监控

**应用场景：**
- 人脸识别（门禁、考勤）
- 行为分析（异常行为检测）
- 人流统计
- 车辆识别（车牌、车型）

**典型案例：**
```
智慧小区门禁系统
- 识别速度：< 0.3 秒
- 准确率：> 99%
- 支持：活体检测（防照片攻击）
```

### 1.2.4 工业检测

**应用场景：**
- 产品缺陷检测
- 尺寸测量
- 零件识别
- 装配验证

**优势：**
- ✅ 24 小时不间断工作
- ✅ 检测精度高（微米级）
- ✅ 一致性高
- ✅ 降低人工成本

### 1.2.5 日常生活

**应用场景：**
- 手机人脸解锁
- 相册自动分类（人物、地点、事物）
- 美颜相机（人脸美化）
- 扫码支付（二维码识别）
- 翻译软件（OCR 文字识别）

## 1.3 计算机视觉的发展历程

### 1.3.1 传统方法阶段（1960s-2010s）

**特点：** 手工设计特征

| 年代 | 代表方法 | 特点 |
|------|---------|------|
| 1960s | 边缘检测 | Roberts 算子 |
| 1970s | 特征提取 | SIFT、SURF |
| 1980s | 形态学 | 腐蚀、膨胀 |
| 1990s | 统计方法 | AdaBoost |
| 2000s | HOG+SVM | 行人检测 |

**典型流程：**
```
图像 → 预处理 → 特征提取 → 分类器 → 结果
          ↓           ↓          ↓
       滤波去噪    HOG/SIFT   SVM/AdaBoost
```

### 1.3.2 深度学习阶段（2012s-至今）

**里程碑事件：**

| 年份 | 事件 | 意义 |
|------|------|------|
| 2012 | AlexNet 赢得 ImageNet | 深度学习元年 |
| 2014 | VGG/GoogLeNet | 网络更深 |
| 2015 | ResNet | 残差连接 |
| 2016 | YOLO | 实时检测 |
| 2017 | Transformer | 注意力机制 |
| 2020 | ViT | Vision Transformer |
| 2022 | Stable Diffusion | 图像生成 |

**准确率提升：**
```
ImageNet 分类准确率：
2010: 72% (传统方法)
2012: 85% (AlexNet)
2015: 96% (ResNet)
2020: 98%+ (EfficientNet)
人类水平：约 95%
```

## 1.4 计算机视觉 vs 其他领域

### 1.4.1 与图像处理的关系

| 维度 | 图像处理 | 计算机视觉 |
|------|---------|-----------|
| **目标** | 改善图像质量 | 理解图像内容 |
| **输入** | 图像 | 图像 |
| **输出** | 图像 | 信息/决策 |
| **例子** | 去噪、增强 | 识别、检测 |

**关系：**
```
图像处理 → 计算机视觉
   ↓           ↓
 手段        目的
```

### 1.4.2 与机器学习的关系

```
人工智能
  └── 机器学习
      └── 深度学习
          └── 计算机视觉（应用领域）
```

**计算机视觉使用的技术：**
- 传统机器学习：SVM、随机森林
- 深度学习：CNN、Transformer
- 其他：聚类、降维
