### 23.2.2 PointNet

```python
import torch
import torch.nn as nn

class PointNet(nn.Module):
    """PointNet 实现"""
    
    def __init__(self, num_classes=10):
        super().__init__()
        
        self.features = nn.Sequential(
            nn.Conv1d(3, 64, 1),
            nn.BatchNorm1d(64),
            nn.ReLU(),
            
            nn.Conv1d(64, 128, 1),
            nn.BatchNorm1d(128),
            nn.ReLU(),
            
            nn.Conv1d(128, 1024, 1),
            nn.BatchNorm1d(1024),
            nn.ReLU()
        )
        
        self.classifier = nn.Sequential(
            nn.Linear(1024, 512),
            nn.ReLU(),
            nn.Linear(512, num_classes)
        )
    
    def forward(self, x):
        # x: [batch, 3, num_points]
        x = self.features(x)
        x = torch.max(x, 2)[0]  # 全局池化
        x = self.classifier(x)
        return x
```

## 23.3 NeRF

### 23.3.1 原理

**NeRF**（Neural Radiance Fields）用神经网络表示 3D 场景。

**输入：** 3D 坐标 + 视角方向  
**输出：** 颜色 + 密度

### 23.3.2 使用

```python
# 使用 nerfacc 库
from nerfacc import NeRF

# 创建模型
nerf = NeRF(
    num_layers=8,
    hidden_dim=256,
    num_views=64
)

# 渲染
rgb, density = nerf(rays_o, rays_d)
```

## 23.4 3D 高斯泼溅

### 23.4.1 原理

**3D Gaussian Splatting**是最新实时渲染技术。

**优势：**
- 实时渲染（100+ FPS）
- 高质量
- 可编辑

### 23.4.2 使用

```python
# 使用 gaussian-splatting 库
from gsplat import GaussianSplatting

# 加载模型
gs = GaussianSplatting.from_pretrained("scene")

# 渲染
image = gs.render(camera_pose)
```

## 23.5 3D 重建

### 23.5.1 SfM

**Structure from Motion**从多视图重建 3D。

```bash
# 使用 COLMAP
colmap feature_extractor --database_path database.db --image_path images/
colmap matcher --database_path database.db
colmap mapper --database_path database.db --image_path images/ --output_path sparse/
```

### 23.5.2 MVS

**Multi-View Stereo**稠密重建。

```bash
# 使用 OpenMVS
OpenMVSInterface sparse/
DensifyPointCloud scene.mvs
ReconstructMesh scene_dense.mvs

TextureMesh scene_dense.mvs
```
