## 24.1 多模态基础

### 24.1.1 什么是多模态

**多模态**结合多种数据类型（文本、图像、音频等）。

**应用：**
- 图文检索
- 视觉问答
- 文生图
- 图像描述

### 24.1.2 多模态对齐

**目标：** 将不同模态映射到同一空间。

```
文本编码器 → 文本嵌入 ─┐
                      ├→ 对比学习 → 对齐空间
图像编码器 → 图像嵌入 ─┘
```

## 24.2 CLIP

### 24.2.1 原理

**CLIP**（Contrastive Language-Image Pre-training）学习图文对齐。

**训练：** 对比学习，最大化匹配图文对的相似度。

### 24.2.2 使用

```python
import clip
import torch
from PIL import Image

# 加载模型
device = "cuda" if torch.cuda.is_available() else "cpu"
model, preprocess = clip.load("ViT-B/32", device=device)

# 处理图像
image = preprocess(Image.open("image.jpg")).unsqueeze(0).to(device)

# 处理文本
texts = clip.tokenize(["a dog", "a cat", "a bird"]).to(device)

# 推理
with torch.no_grad():
    image_features = model.encode_image(image)
    text_features = model.encode_text(texts)
    
    # 计算相似度
    similarity = (image_features @ text_features.T).softmax(dim=-1)

print(f"相似度：{similarity}")
```

### 24.2.3 应用：零样本分类

```python
import clip
import torch
from PIL import Image

# 加载模型
model, preprocess = clip.load("ViT-B/32")

# 准备类别
classes = ["dog", "cat", "bird", "car", "plane"]
texts = clip.tokenize([f"a photo of a {c}" for c in classes])

# 处理图像
image = preprocess(Image.open("test.jpg")).unsqueeze(0)

# 推理
with torch.no_grad():
    image_features = model.encode_image(image)
    text_features = model.encode_text(texts)
    
    # 相似度
    similarity = (image_features @ text_features.T).softmax(dim=-1)
    predicted_class = classes[similarity.argmax()]

print(f"预测类别：{predicted_class}")
```

## 24.6 实战：图文检索系统

### 24.6.1 系统架构

```
图像库 → CLIP 图像编码 → 图像嵌入向量库
                            ↓
用户文本 → CLIP 文本编码 → 相似度检索 → 返回图像
```

### 24.6.2 实现

```python
import clip
import torch
from PIL import Image
import faiss
import numpy as np

class ImageSearchEngine:
    """图文检索引擎"""
    
    def __init__(self):
        self.device = "cuda" if torch.cuda.is_available() else "cpu"
        self.model, self.preprocess = clip.load("ViT-B/32", self.device)
        self.index = None
        self.images = []
    
    def build_index(self, image_paths):
        """构建索引"""
        self.images = image_paths
        embeddings = []
        
        for path in image_paths:
            image = self.preprocess(Image.open(path)).unsqueeze(0).to(self.device)
            with torch.no_grad():
                embedding = self.model.encode_image(image)
            embeddings.append(embedding.cpu().numpy())
        
        embeddings = np.vstack(embeddings).astype('float32')
        
        # 归一化
        faiss.normalize_L2(embeddings)
        
        # 构建 FAISS 索引
        self.index = faiss.IndexFlatIP(embeddings.shape[1])
        self.index.add(embeddings)
    
    def search(self, query, top_k=5):
        """搜索图像"""
        # 文本编码
        text = clip.tokenize([query]).to(self.device)
        with torch.no_grad():
            text_embedding = self.model.encode_text(text)
        text_embedding = text_embedding.cpu().numpy().astype('float32')
        faiss.normalize_L2(text_embedding)
        
        # 检索
        distances, indices = self.index.search(text_embedding, top_k)
        
        results = []
        for idx, dist in zip(indices[0], distances[0]):
            results.append({
                'image_path': self.images[idx],
                'similarity': dist
            })
        
        return results

# 使用
engine = ImageSearchEngine()
engine.build_index(['img1.jpg', 'img2.jpg', 'img3.jpg'])
results = engine.search("a cute dog", top_k=3)

for result in results:
    print(f"{result['image_path']}: {result['similarity']:.4f}")
```
