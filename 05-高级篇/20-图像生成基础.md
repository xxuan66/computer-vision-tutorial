## 20.1 生成模型概述

### 20.1.1 什么是生成模型

**生成模型**学习数据的分布，生成新的样本。

**vs 判别模型：**
- 判别模型：P(y|x) - 判断类别
- 生成模型：P(x) - 生成新样本

### 20.1.2 应用场景

| 应用 | 说明 |
|------|------|
| **图像生成** | 生成逼真图像 |
| **风格迁移** | 艺术风格转换 |
| **图像修复** | 填补缺失区域 |
| **超分辨率** | 提升图像分辨率 |
| **数据增强** | 生成训练数据 |

## 20.2 GAN 原理

### 20.2.1 生成对抗网络

**GAN**（Generative Adversarial Network）由两个网络组成：

```
生成器（Generator）     判别器（Discriminator）
     ↓                        ↓
随机噪声 → 生成图像    真实/生成图像 → 真/假判断
     ↓                        ↓
  欺骗判别器              准确判断
```

**对抗过程：**
- 生成器：尽量生成逼真图像，欺骗判别器
- 判别器：尽量准确判断真假

### 20.2.2 损失函数

```python
import torch
import torch.nn as nn

# 判别器损失
def discriminator_loss(real_output, fake_output):
    real_loss = nn.BCELoss()(real_output, torch.ones_like(real_output))
    fake_loss = nn.BCELoss()(fake_output, torch.zeros_like(fake_output))
    return real_loss + fake_loss

# 生成器损失
def generator_loss(fake_output):
    return nn.BCELoss()(fake_output, torch.ones_like(fake_output))
```

## 20.3 DCGAN

### 20.3.1 架构

**DCGAN**（Deep Convolutional GAN）使用卷积网络。

**生成器：**
```
随机噪声 → 转置卷积 → ReLU → 转置卷积 → Tanh → 生成图像
```

**判别器：**
```
图像 → 卷积 → LeakyReLU → 卷积 → Sigmoid → 真/假
```

### 20.3.2 实现

```python
import torch
import torch.nn as nn

class Generator(nn.Module):
    """DCGAN 生成器"""
    
    def __init__(self, nz=100, ngf=64, nc=3):
        super().__init__()
        
        self.main = nn.Sequential(
            nn.ConvTranspose2d(nz, ngf * 8, 4, 1, 0, bias=False),
            nn.BatchNorm2d(ngf * 8),
            nn.ReLU(True),
            
            nn.ConvTranspose2d(ngf * 8, ngf * 4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 4),
            nn.ReLU(True),
            
            nn.ConvTranspose2d(ngf * 4, ngf * 2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 2),
            nn.ReLU(True),
            
            nn.ConvTranspose2d(ngf * 2, ngf, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf),
            nn.ReLU(True),
            
            nn.ConvTranspose2d(ngf, nc, 4, 2, 1, bias=False),
            nn.Tanh()
        )
    
    def forward(self, input):
        return self.main(input)

class Discriminator(nn.Module):
    """DCGAN 判别器"""
    
    def __init__(self, nc=3, ndf=64):
        super().__init__()
        
        self.main = nn.Sequential(
            nn.Conv2d(nc, ndf, 4, 2, 1, bias=False),
            nn.LeakyReLU(0.2, inplace=True),
            
            nn.Conv2d(ndf, ndf * 2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ndf * 2),
            nn.LeakyReLU(0.2, inplace=True),
            
            nn.Conv2d(ndf * 2, ndf * 4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ndf * 4),
            nn.LeakyReLU(0.2, inplace=True),
            
            nn.Conv2d(ndf * 4, ndf * 8, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ndf * 8),
            nn.LeakyReLU(0.2, inplace=True),
            
            nn.Conv2d(ndf * 8, 1, 4, 1, 0, bias=False),
            nn.Sigmoid()
        )
    
    def forward(self, input):
        return self.main(input)
```

## 20.4 StyleGAN

### 20.4.1 原理

**StyleGAN**引入风格控制，生成高质量图像。

**创新点：**
- 映射网络（Mapping Network）
- 自适应实例归一化（AdaIN）
- 风格混合

### 20.4.2 使用预训练模型

```python
import torch
from stylegan2_pytorch import StyleGAN2

# 加载预训练模型
model = StyleGAN2(
    image_size = 1024,
    latent_dim = 512
)

# 生成图像
latent = torch.randn(1, 512)
generated_image = model(latent)
```

## 20.5 训练技巧

### 20.5.1 训练稳定性

```python
# 标签平滑
real_labels = torch.ones(batch_size, 1) * 0.9
fake_labels = torch.zeros(batch_size, 1)

# 学习率
optimizer_G = torch.optim.Adam(generator.parameters(), lr=0.0002, betas=(0.5, 0.999))
optimizer_D = torch.optim.Adam(discriminator.parameters(), lr=0.0002, betas=(0.5, 0.999))
```

### 20.5.2 训练循环

```python
for epoch in range(num_epochs):
    for i, (real_images, _) in enumerate(dataloader):
        # 训练判别器
        discriminator.zero_grad()
        
        real_output = discriminator(real_images)
        fake_images = generator(torch.randn(batch_size, nz))
        fake_output = discriminator(fake_images.detach())
        
        d_loss = discriminator_loss(real_output, fake_output)
        d_loss.backward()
        optimizer_D.step()
        
        # 训练生成器
        generator.zero_grad()
        
        fake_images = generator(torch.randn(batch_size, nz))
        fake_output = discriminator(fake_images)
        
        g_loss = generator_loss(fake_output)
        g_loss.backward()
        optimizer_G.step()
    
    print(f"Epoch {epoch+1}, D Loss: {d_loss.item():.4f}, G Loss: {g_loss.item():.4f}")
```
