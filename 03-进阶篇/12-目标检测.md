## 12.1 目标检测基础

### 12.1.1 什么是目标检测

**目标检测**既要识别物体是什么，又要定位物体在哪里。

**输出：**
- 类别标签（是什么）
- 边界框（在哪里）

### 12.1.2 vs 图像分类

| 任务 | 输入 | 输出 |
|------|------|------|
| **图像分类** | 图像 | 类别标签 |
| **目标检测** | 图像 | 类别 + 边界框 |

### 12.1.3 评估指标

| 指标 | 说明 |
|------|------|
| **IoU** | 预测框与真实框的交并比 |
| **mAP** | 平均精度均值 |
| **FPS** | 每秒检测帧数 |

## 12.2 R-CNN 系列

### 12.2.1 R-CNN（2014）

**两阶段检测：候选区域 + CNN 分类**

**流程：**
```
输入图像 → 选择性搜索 → 候选框 → CNN 特征 → SVM 分类 → 边界框回归
```

**缺点：**
- 速度慢（~20 秒/图）
- 训练复杂

### 12.2.2 Fast R-CNN（2015）

**改进：**
- 共享卷积计算
- 多任务损失

**速度：** ~2 秒/图

### 12.2.3 Faster R-CNN（2015）

**改进：**
- 区域提议网络（RPN）
- 端到端训练

**速度：** ~7 FPS

## 12.3 YOLO 系列

### 12.3.1 YOLO 原理

**You Only Look Once** — 单阶段检测

**核心思想：**
> 将检测视为回归问题，直接从图像到边界框

**特点：**
- ✅ 速度极快
- ✅ 全局理解
- ❌ 小物体检测差

### 12.3.2 使用 YOLOv5

```python
import torch

# 加载 YOLOv5 模型
model = torch.hub.load('ultralytics/yolov5', 'yolov5s')

# 检测
img = 'image.jpg'
results = model(img)

# 显示结果
results.show()

# 获取检测结果
df = results.pandas().xyxy[0]
print(df)
```

## 12.4 SSD

### 12.4.1 原理

**Single Shot MultiBox Detector**

**特点：**
- 单阶段检测
- 多尺度特征图
- 默认框（Default Boxes）

### 12.4.2 使用 SSD

```python
import torch
import torchvision.models as models

# 加载预训练 SSD
model = models.detection.ssdlite320_mobilenet_v3_large(pretrained=True)
model.eval()

# 检测
img = torch.rand(1, 3, 300, 300)
with torch.no_grad():
    prediction = model(img)

print(f"检测到 {len(prediction[0]['boxes'])} 个物体")
```

## 12.5 实战：自定义目标检测

### 12.5.1 数据准备

```python
import torch
from torch.utils.data import Dataset
import cv2

class CustomDataset(Dataset):
    """自定义检测数据集"""
    
    def __init__(self, annotations, img_dir, transform=None):
        self.annotations = annotations
        self.img_dir = img_dir
        self.transform = transform
    
    def __len__(self):
        return len(self.annotations)
    
    def __getitem__(self, idx):
        # 加载图像
        img_path = self.annotations[idx]['image_path']
        img = cv2.imread(img_path)
        img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
        
        # 加载标注
        boxes = torch.as_tensor(self.annotations[idx]['boxes'], dtype=torch.float32)
        labels = torch.as_tensor(self.annotations[idx]['labels'], dtype=torch.int64)
        
        target = {}
        target['boxes'] = boxes
        target['labels'] = labels
        
        if self.transform:
            img = self.transform(img)
        
        return img, target
```

### 12.5.2 模型训练

```python
import torch
import torchvision
from torchvision.models.detection import fasterrcnn_resnet50_fpn

# 加载模型
model = fasterrcnn_resnet50_fpn(pretrained=True)

# 替换分类头
num_classes = 10  # 自定义类别数
in_features = model.roi_heads.box_predictor.cls_score.in_features
model.roi_heads.box_predictor = torchvision.models.detection.faster_rcnn.FastRCNNPredictor(in_features, num_classes)

# 训练配置
params = [p for p in model.parameters() if p.requires_grad]
optimizer = torch.optim.SGD(params, lr=0.005, momentum=0.9, weight_decay=0.0005)
```
