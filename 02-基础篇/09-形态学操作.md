## 9.1 形态学基础

### 9.1.1 什么是形态学

**数学形态学**是一类基于形状特征的图像处理方法。

**核心思想：**
> 用结构元素探测图像，提取有用的形状信息

**基本操作：**
- 腐蚀（Erosion）
- 膨胀（Dilation）
- 开运算（Opening）
- 闭运算（Closing）

### 9.1.2 结构元素

**结构元素**可以理解成形态学操作里的“探针”。

```python
import cv2
import numpy as np

# 矩形结构元素
kernel_rect = cv2.getStructuringElement(cv2.MORPH_RECT, (5, 5))

# 椭圆形结构元素
kernel_ellipse = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5, 5))

# 十字形结构元素
kernel_cross = cv2.getStructuringElement(cv2.MORPH_CROSS, (5, 5))

print("矩形核：")
print(kernel_rect)
print("\n椭圆核：")
print(kernel_ellipse)
print("\n十字核：")
print(kernel_cross)
```

## 9.2 腐蚀操作

### 9.2.1 原理

**腐蚀**使图像中的高亮区域缩小。

**数学表示：**
```
A ⊖ B = {z | Bz ⊆ A}

其中：
- A: 输入图像
- B: 结构元素
- ⊖: 腐蚀操作
```

### 9.2.2 代码实现

```python
import cv2
import numpy as np

# 读取图像（二值）
img = cv2.imread('image.png', cv2.IMREAD_GRAYSCALE)
_, img = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

# 结构元素
kernel = np.ones((5, 5), np.uint8)

# 腐蚀
erosion = cv2.erode(img, kernel, iterations=1)

# 多次腐蚀
erosion_2 = cv2.erode(img, kernel, iterations=2)
erosion_3 = cv2.erode(img, kernel, iterations=3)

# 显示
cv2.imshow('Original', img)
cv2.imshow('Erosion 1x', erosion)
cv2.imshow('Erosion 2x', erosion_2)
cv2.imshow('Erosion 3x', erosion_3)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 9.2.3 应用场景

| 应用 | 说明 |
|------|------|
| **去除噪声** | 消除小亮点 |
| **分离物体** | 分离粘连物体 |
| **细化线条** | 使线条变细 |

## 9.3 膨胀操作

### 9.3.1 原理

**膨胀**使图像中的高亮区域扩大。

**数学表示：**
```
A ⊕ B = {z | (Bz)^c ∩ A ≠ ∅}

其中：
- A: 输入图像
- B: 结构元素
- ⊕: 膨胀操作
```

### 9.3.2 代码实现

```python
import cv2
import numpy as np

# 读取图像（二值）
img = cv2.imread('image.png', cv2.IMREAD_GRAYSCALE)
_, img = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

# 结构元素
kernel = np.ones((5, 5), np.uint8)

# 膨胀
dilation = cv2.dilate(img, kernel, iterations=1)

# 多次膨胀
dilation_2 = cv2.dilate(img, kernel, iterations=2)
dilation_3 = cv2.dilate(img, kernel, iterations=3)

# 显示
cv2.imshow('Original', img)
cv2.imshow('Dilation 1x', dilation)
cv2.imshow('Dilation 2x', dilation_2)
cv2.imshow('Dilation 3x', dilation_3)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 9.3.3 应用场景

| 应用 | 说明 |
|------|------|
| **填充空洞** | 填补物体内部空洞 |
| **连接物体** | 连接断裂部分 |
| **粗化线条** | 使线条变粗 |

## 9.4 开运算与闭运算

### 9.4.1 开运算

**开运算** = 先腐蚀后膨胀

**作用：**
- 去除小物体
- 平滑边界
- 断开狭窄连接

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.png', cv2.IMREAD_GRAYSCALE)
_, img = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

# 结构元素
kernel = np.ones((5, 5), np.uint8)

# 开运算
opening = cv2.morphologyEx(img, cv2.MORPH_OPEN, kernel)

# 显示
cv2.imshow('Original', img)
cv2.imshow('Opening', opening)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 9.4.2 闭运算

**闭运算** = 先膨胀后腐蚀

**作用：**
- 填充小空洞
- 连接邻近物体
- 平滑边界

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.png', cv2.IMREAD_GRAYSCALE)
_, img = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

# 结构元素
kernel = np.ones((5, 5), np.uint8)

# 闭运算
closing = cv2.morphologyEx(img, cv2.MORPH_CLOSE, kernel)

# 显示
cv2.imshow('Original', img)
cv2.imshow('Closing', closing)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 9.4.3 开闭运算对比

| 操作 | 顺序 | 作用 | 应用 |
|------|------|------|------|
| **开运算** | 腐蚀→膨胀 | 去除小物体 | 去噪 |
| **闭运算** | 膨胀→腐蚀 | 填充小空洞 | 补洞 |

## 9.7 实战应用

### 9.7.1 文字提取

```python
import cv2
import numpy as np

def extract_text(img_path):
    """提取图像中的文字"""
    # 读取图像
    img = cv2.imread(img_path, cv2.IMREAD_GRAYSCALE)
    
    # 二值化
    _, binary = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)
    
    # 形态学操作（提取水平线）
    horizontal_kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (40, 1))
    horizontal_lines = cv2.morphologyEx(binary, cv2.MORPH_OPEN, horizontal_kernel, iterations=2)
    
    # 形态学操作（提取垂直线）
    vertical_kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (1, 40))
    vertical_lines = cv2.morphologyEx(binary, cv2.MORPH_OPEN, vertical_kernel, iterations=2)
    
    # 合并线条
    lines = cv2.addWeighted(horizontal_lines, 0.5, vertical_lines, 0.5, 0)
    
    # 提取文字区域
    text_region = cv2.subtract(binary, lines)
    
    return text_region

# 使用
text = extract_text('document.png')
cv2.imshow('Text', text)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 9.7.2 缺陷检测

```python
import cv2
import numpy as np

def detect_defects(img_path):
    """检测产品表面缺陷"""
    # 读取图像
    img = cv2.imread(img_path, cv2.IMREAD_GRAYSCALE)
    
    # 闭运算（填充正常区域）
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (15, 15))
    closed = cv2.morphologyEx(img, cv2.MORPH_CLOSE, kernel)
    
    # 差分（提取缺陷）
    diff = cv2.absdiff(img, closed)
    
    # 二值化
    _, defects = cv2.threshold(diff, 30, 255, cv2.THRESH_BINARY)
    
    # 形态学清理
    kernel = np.ones((3, 3), np.uint8)
    defects = cv2.dilate(defects, kernel, iterations=2)
    defects = cv2.erode(defects, kernel, iterations=1)
    
    return defects

# 使用
defects = detect_defects('product.png')
cv2.imshow('Defects', defects)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 9.7.3 细胞计数

```python
import cv2
import numpy as np

def count_cells(img_path):
    """细胞计数"""
    # 读取图像
    img = cv2.imread(img_path, cv2.IMREAD_GRAYSCALE)
    
    # 二值化
    _, binary = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)
    
    # 开运算（分离粘连细胞）
    kernel = np.ones((3, 3), np.uint8)
    opening = cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel, iterations=2)
    
    # 膨胀（确保细胞连接）
    sure_bg = cv2.dilate(opening, kernel, iterations=3)
    
    # 距离变换
    dist_transform = cv2.distanceTransform(opening, cv2.DIST_L2, 5)
    _, sure_fg = cv2.threshold(dist_transform, 0.7 * dist_transform.max(), 255, 0)
    
# 查找轮廓
    contours, _ = cv2.findContours(sure_fg.astype(np.uint8), cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    # 计数
    cell_count = len(contours)
    
    # 绘制结果
    result = cv2.cvtColor(opening, cv2.COLOR_GRAY2BGR)
    cv2.drawContours(result, contours, -1, (0, 255, 0), 2)
    
    return result, cell_count

# 使用
result, count = count_cells('cells.png')
print(f"细胞数量：{count}")

cv2.imshow('Cells', result)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 9.8 本章小结

### 核心知识点

1. **腐蚀** — 缩小高亮区域
2. **膨胀** — 扩大高亮区域
3. **开运算** — 去噪、分离
4. **闭运算** — 填充、连接
5. **形态学梯度** — 边缘提取
6. **顶帽/黑帽** — 亮点/暗点提取
