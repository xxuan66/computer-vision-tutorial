## 5.1 图像的本质

### 5.1.1 什么是数字图像

**数字图像**可以先理解成由像素组成的二维矩阵，每个像素代表图像中的一个点。

**数学表示：**
```
I(x, y) = 像素值

其中：
- x, y: 像素坐标
- I: 像素强度（灰度或颜色）
```

**示例：**
```
一个 256x256 的灰度图像：
- 宽度：256 像素
- 高度：256 像素
- 总像素数：256 × 256 = 65,536
- 每个像素值：0-255（8 位）
```

### 5.1.2 图像类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **二值图像** | 只有 0 和 255 | 文字、签名 |
| **灰度图像** | 0-255 灰度级 | 黑白照片 |
| **彩色图像** | RGB 三通道 | 彩色照片 |
| **多光谱图像** | 多个波段 | 卫星图像 |

## 5.2 颜色空间

### 5.2.1 RGB 颜色空间

**RGB**（Red, Green, Blue）是最常用的颜色空间。

**特点：**
- 三原色：红、绿、蓝
- 加色模型（叠加变亮）
- 每个通道 8 位（0-255）
- 总颜色数：256³ = 16,777,216

**代码示例：**
```python
import cv2
import numpy as np

# 创建纯色图像
img_red = np.zeros((100, 100, 3), dtype=np.uint8)
img_red[:, :] = [0, 0, 255]  # BGR 格式

img_green = np.zeros((100, 100, 3), dtype=np.uint8)
img_green[:, :] = [0, 255, 0]

img_blue = np.zeros((100, 100, 3), dtype=np.uint8)
img_blue[:, :] = [255, 0, 0]

# 显示
cv2.imshow('Red', img_red)
cv2.imshow('Green', img_green)
cv2.imshow('Blue', img_blue)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 5.2.2 HSV 颜色空间

**HSV**（Hue, Saturation, Value）更符合人类对颜色的感知。

| 分量 | 说明 | 范围 |
|------|------|------|
| **H（色调）** | 颜色类型 | 0-180° |
| **S（饱和度）** | 颜色纯度 | 0-255 |
| **V（明度）** | 亮度 | 0-255 |

**应用场景：**
- 颜色分割
- 图像检索
- 颜色匹配

**代码示例：**
```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg')

# 转换为 HSV
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 分离通道
h, s, v = cv2.split(hsv)

# 显示
cv2.imshow('Hue', h)
cv2.imshow('Saturation', s)
cv2.imshow('Value', v)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 5.2.3 LAB 颜色空间

**LAB**（Lightness, A, B）是感知均匀的颜色空间。

| 分量 | 说明 |
|------|------|
| **L（亮度）** | 明暗程度（0-100） |
| **A（绿 - 红）** | 绿色到红色（-128 到 127） |
| **B（蓝 - 黄）** | 蓝色到黄色（-128 到 127） |

**应用场景：**
- 颜色校正
- 图像增强
- 肤色检测

### 5.2.4 颜色空间对比

| 特性 | RGB | HSV | LAB |
|------|-----|-----|-----|
| **直观性** | ⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| **计算效率** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐ |
| **颜色分割** | ⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| **设备依赖** | 是 | 是 | 否 |

## 5.3 图像直方图

### 5.3.1 什么是直方图

**直方图**显示了图像中像素强度的分布情况。

**数学表示：**
```
H(i) = 像素值为 i 的像素数量

其中 i = 0, 1, 2, ..., 255
```

### 5.3.2 计算直方图

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

# 读取图像（灰度）
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# 计算直方图
hist = cv2.calcHist([img], [0], None, [256], [0, 256])

# 绘制直方图
plt.figure(figsize=(10, 6))
plt.plot(hist, color='black')
plt.xlabel('Pixel Value')
plt.ylabel('Frequency')
plt.title('Histogram')
plt.xlim([0, 256])
plt.grid(alpha=0.3)
plt.show()
```

### 5.3.3 彩色图像直方图

```python
import cv2
import matplotlib.pyplot as plt

# 读取图像（RGB）
img = cv2.imread('image.jpg')
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

# 计算各通道直方图
colors = ('r', 'g', 'b')
channels = (2, 1, 0)  # BGR → RGB

plt.figure(figsize=(10, 6))

for i, (color, channel) in enumerate(zip(colors, channels)):
    hist = cv2.calcHist([img], [channel], None, [256], [0, 256])
    plt.plot(hist, color=color)

plt.xlabel('Pixel Value')
plt.ylabel('Frequency')
plt.title('Color Histogram')
plt.xlim([0, 256])
plt.grid(alpha=0.3)
plt.show()
```

### 5.3.4 直方图应用

**1. 图像增强（直方图均衡化）**
```python
import cv2

# 读取灰度图像
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# 直方图均衡化
img_equalized = cv2.equalizeHist(img)

# 显示
cv2.imshow('Original', img)
cv2.imshow('Equalized', img_equalized)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

**2. 对比度增强（CLAHE）**
```python
import cv2

# 读取图像
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# 创建 CLAHE 对象
clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))

# 应用 CLAHE
img_clahe = clahe.apply(img)

# 显示
cv2.imshow('Original', img)
cv2.imshow('CLAHE', img_clahe)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 5.4 图像通道操作

### 5.4.1 通道分离与合并

```python
import cv2

# 读取图像
img = cv2.imread('image.jpg')

# 分离通道
b, g, r = cv2.split(img)

# 合并通道
img_merged = cv2.merge([b, g, r])

# 只显示红色通道
img_red_only = cv2.merge([b*0, g*0, r])

# 显示
cv2.imshow('Blue', b)
cv2.imshow('Green', g)
cv2.imshow('Red', r)
cv2.imshow('Merged', img_merged)
cv2.imshow('Red Only', img_red_only)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 5.4.2 通道操作应用

**提取特定颜色：**
```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg')
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 定义蓝色范围
lower_blue = np.array([100, 50, 50])
upper_blue = np.array([130, 255, 255])

# 创建掩码
mask = cv2.inRange(hsv, lower_blue, upper_blue)

# 应用掩码
result = cv2.bitwise_and(img, img, mask=mask)

# 显示
cv2.imshow('Original', img)
cv2.imshow('Mask', mask)
cv2.imshow('Result', result)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 5.5 图像金字塔

### 5.5.1 高斯金字塔

**高斯金字塔**用于图像缩放和多尺度分析。

```python
import cv2

# 读取图像
img = cv2.imread('image.jpg')

# 下采样（缩小）
img_lower = cv2.pyrDown(img)

# 上采样（放大）
img_higher = cv2.pyrUp(img)

# 多级金字塔
images = [img]
for i in range(5):
    img = cv2.pyrDown(img)
    images.append(img)

# 显示
for i, image in enumerate(images):
    cv2.imshow(f'Level {i}', image)

cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 5.5.2 应用场景

| 应用 | 说明 |
|------|------|
| **图像融合** | 多尺度融合 |
| **目标检测** | 多尺度检测 |
| **图像压缩** | 渐进式传输 |

## 5.6 插值方法

### 5.6.1 常见插值方法

| 方法 | 说明 | 速度 | 质量 |
|------|------|------|------|
| **最近邻** | 取最近像素 | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| **双线性** | 4 邻域插值 | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| **双三次** | 16 邻域插值 | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| **Lanczos** | 高质量插值 | ⭐⭐ | ⭐⭐⭐⭐⭐ |

### 5.6.2 代码示例

```python
import cv2

# 读取图像
img = cv2.imread('image.jpg')

# 不同插值方法缩放
img_nearest = cv2.resize(img, None, fx=2, fy=2, interpolation=cv2.INTER_NEAREST)
img_linear = cv2.resize(img, None, fx=2, fy=2, interpolation=cv2.INTER_LINEAR)
img_cubic = cv2.resize(img, None, fx=2, fy=2, interpolation=cv2.INTER_CUBIC)
img_lanczos = cv2.resize(img, None, fx=2, fy=2, interpolation=cv2.INTER_LANCZOS4)

# 显示
cv2.imshow('Nearest', img_nearest)
cv2.imshow('Linear', img_linear)
cv2.imshow('Cubic', img_cubic)
cv2.imshow('Lanczos', img_lanczos)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
