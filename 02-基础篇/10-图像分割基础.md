## 10.1 图像分割概述

### 10.1.1 什么是图像分割

**图像分割**是将图像划分为多个有意义区域的过程。

**目标：**
> 简化图像表示，使其更容易分析和理解

**应用：**
- 医学影像（肿瘤分割）
- 自动驾驶（道路分割）
- 工业检测（缺陷定位）
- 人脸识别（面部区域提取）

### 10.1.2 分割方法分类

| 类别 | 方法 | 特点 |
|------|------|------|
| **基于阈值** | 全局阈值、自适应阈值 | 简单快速 |
| **基于边缘** | Canny、Sobel | 边缘精确 |
| **基于区域** | 区域生长、分裂合并 | 区域完整 |
| **基于聚类** | K-means、Mean Shift | 自动分类 |
| **基于深度学习** | FCN、U-Net | 精度最高 |

## 10.3 区域生长

### 10.3.1 原理

**区域生长**从种子点开始，将相似像素合并到同一区域。

**步骤：**
1. 选择种子点
2. 检查邻域像素
3. 合并相似像素
4. 重复直到收敛

### 10.3.2 代码实现

```python
import cv2
import numpy as np

def region_growing(img, seed_point, threshold=10):
    """区域生长算法"""
    rows, cols = img.shape
    segmented = np.zeros_like(img)
    
    # 种子点值
    seed_value = img[seed_point]
    
    # 待处理队列
    queue = [seed_point]
    segmented[seed_point] = 255
    
    while queue:
        x, y = queue.pop(0)
        
        # 检查 8 邻域
        for dx in [-1, 0, 1]:
            for dy in [-1, 0, 1]:
                if dx == 0 and dy == 0:
                    continue
                
                nx, ny = x + dx, y + dy
                
                # 边界检查
                if 0 <= nx < rows and 0 <= ny < cols:
                    if segmented[nx, ny] == 0:
                        # 相似性检查
                        if abs(int(img[nx, ny]) - int(seed_value)) <= threshold:
                            segmented[nx, ny] = 255
                            queue.append((nx, ny))
    
    return segmented

# 使用
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)
seed = (100, 100)  # 种子点
result = region_growing(img, seed, threshold=15)

cv2.imshow('Original', img)
cv2.imshow('Segmented', result)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 10.4 分水岭算法

### 10.4.1 原理

**分水岭算法**将图像视为地形，水从低点淹没到高点。

**应用：**
- 物体分离
- 细胞分割
- 图像分割

### 10.4.2 代码实现

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 二值化
_, thresh = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)

# 噪声去除
kernel = np.ones((3, 3), np.uint8)
opening = cv2.morphologyEx(thresh, cv2.MORPH_OPEN, kernel, iterations=2)

# 确定背景区域
sure_bg = cv2.dilate(opening, kernel, iterations=3)

# 确定前景区域
dist_transform = cv2.distanceTransform(opening, cv2.DIST_L2, 5)
_, sure_fg = cv2.threshold(dist_transform, 0.7 * dist_transform.max(), 255, 0)

# 找到未知区域
sure_fg = np.uint8(sure_fg)
unknown = cv2.subtract(sure_bg, sure_fg)

# 标记
_, markers = cv2.connectedComponents(sure_fg)
markers = markers + 1
markers[unknown == 255] = 0

# 应用分水岭
markers = cv2.watershed(img, markers)
img[markers == -1] = [0, 0, 255]  # 边界标记为红色

# 显示
cv2.imshow('Original', img)
cv2.imshow('Segmentation', markers.astype(np.uint8))
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 10.5 K-means 分割

### 10.5.1 原理

**K-means**将像素聚类到 K 个簇中。

**步骤：**
1. 随机初始化 K 个中心
2. 分配像素到最近中心
3. 更新中心位置
4. 重复直到收敛

### 10.5.2 代码实现

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg')
Z = img.reshape((-1, 3))
Z = np.float32(Z)

# 定义标准
criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 10, 1.0)

# 设置 K 值
K = 3
flags = cv2.KMEANS_RANDOM_CENTERS

# K-means
compactness, labels, centers = cv2.kmeans(Z, K, None, criteria, 10, flags)

# 转换回 uint8
centers = np.uint8(centers)
res = centers[labels.flatten()]
res = res.reshape(img.shape)

# 显示
cv2.imshow('Original', img)
cv2.imshow('K-means', res)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### 10.5.3 应用：图像压缩

```python
import cv2
import numpy as np

def compress_image(img_path, K=16):
    """使用 K-means 压缩图像"""
    # 读取图像
    img = cv2.imread(img_path)
    Z = img.reshape((-1, 3))
    Z = np.float32(Z)
    
    # K-means
    criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 10, 1.0)
    flags = cv2.KMEANS_RANDOM_CENTERS
    compactness, labels, centers = cv2.kmeans(Z, K, None, criteria, 10, flags)
    
    # 重建图像
    centers = np.uint8(centers)
    compressed = centers[labels.flatten()]
    compressed = compressed.reshape(img.shape)
    
    return compressed

# 使用
compressed = compress_image('image.jpg', K=16)
cv2.imwrite('compressed.jpg', compressed)

cv2.imshow('Original', cv2.imread('image.jpg'))
cv2.imshow('Compressed', compressed)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 10.6 GrabCut 算法

### 10.6.1 原理

**GrabCut**是交互式前景提取算法。

**特点：**
- 用户指定矩形框
- 迭代优化
- 精确分割

### 10.6.2 代码实现

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg')
mask = np.zeros(img.shape[:2], np.uint8)

# GrabCut 参数
bgdModel = np.zeros((1, 65), np.float64)
fgdModel = np.zeros((1, 65), np.float64)

# 定义矩形（前景区域）
rect = (50, 50, 450, 290)

# 应用 GrabCut
cv2.grabCut(img, mask, rect, bgdModel, fgdModel, 5, cv2.GC_INIT_WITH_RECT)

# 生成掩码
mask2 = np.where((mask == 2) | (mask == 0), 0, 1).astype('uint8')

# 提取前景
foreground = img * mask2[:, :, np.newaxis]

# 显示
cv2.imshow('Original', img)
cv2.imshow('Foreground', foreground)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 10.7 分割评估

### 10.7.1 评估指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **准确率** | (TP+TN)/(TP+TN+FP+FN) | 总体准确率 |
| **精确率** | TP/(TP+FP) | 查准率 |
| **召回率** | TP/(TP+FN) | 查全率 |
| **IoU** | TP/(TP+FP+FN) | 交并比 |
| **Dice** | 2TP/(2TP+FP+FN) | 相似系数 |

### 10.7.2 代码实现

```python
import numpy as np

def calculate_iou(pred, gt):
    """计算 IoU"""
    intersection = np.logical_and(pred, gt)
    union = np.logical_or(pred, gt)
    iou = np.sum(intersection) / np.sum(union)
    return iou

def calculate_dice(pred, gt):
    """计算 Dice 系数"""
    intersection = np.logical_and(pred, gt)
    dice = 2 * np.sum(intersection) / (np.sum(pred) + np.sum(gt))
    return dice

# 使用示例
pred = np.array([1, 1, 0, 1, 0, 1, 0, 0])
gt = np.array([1, 1, 0, 0, 0, 1, 1, 0])

iou = calculate_iou(pred, gt)
dice = calculate_dice(pred, gt)

print(f"IoU: {iou:.4f}")
print(f"Dice: {dice:.4f}")
```
